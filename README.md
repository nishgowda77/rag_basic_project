 🤖 AI PDF Chatbot

An AI-powered PDF chatbot built using Python and Streamlit that allows users to upload PDF files and ask questions interactively.

---

 📌 Overview

This project helps users interact with PDF documents using AI.  
Users can upload files and search answers from the document content.

---

✨ Features

 📄 Upload PDF files
 💬 Ask questions from documents
 ⚡ Fast responses
 🖥️ Streamlit user interface
 🤖 AI-powered chatbot

---

▶️ Run Locally

```bash
pip install -r requirements.txt
streamlit run app.py
```

---

 🐳 Run with Docker

```bash
docker build -t ai-pdf-chatbot .
docker run -p 8501:8501 ai-pdf-chatbot
```
